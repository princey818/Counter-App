# Counter App

A simple and beginner-friendly **Counter Application** built using **Angular**. This project allows you to increment, decrement, and reset a counter with a clean and responsive UI. It has a simple Angular code structure, making it perfect for beginners, and optionally supports LocalStorage to save the counter value.

## Features
- Increment, decrement, and reset counter
- Clean and responsive UI
- Simple Angular code structure
- Optional LocalStorage support

## How to Run the Project
To run the project, first clone the repository and navigate to the project folder. Install all dependencies using `npm install` and then start the Angular development server with `ng serve`. Once the server is running, open your browser and go to `http://localhost:4200/` to see the counter app in action.

## Project Structure
The project is organized as follows: `counter.ts` contains the counter logic (increment, decrement, reset), `counter.html` defines the UI layout, `counter.css` handles the styling, and `app.component.ts` along with `app.routes.ts` manage the routing and setup.

## Highlights
The app offers a simple and clean UI, is fast and responsive, provides fully functional counter buttons, and is ideal for Angular beginners.

## Build the Project
To create a production build of the project, simply run `ng build`.
